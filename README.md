# next-exhibition-config
전시 + 결 코드


## "전시 + 결" 이란?
---
전시와 사람 사이의 '인연(결)' 이라는 뜻으로 전시 관람을 위한 모임 사이트 입니다. 


## 웹의 기능
---
1.  로그인, 회원가입, 아이디찾기, 비밀번호 찾기

2.  모임 생성, 삭제 : 본인이 모임을 생성 하고 수정,삭제가 가능 합니다.
   

4.  모임 참석/취소 : 생성된 모임에 참석/참석취소 가 가능합니다.
